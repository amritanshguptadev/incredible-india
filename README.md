# Incredible India — Travel Guide

A handcrafted travel portal showcasing India's most iconic destinations, built with pure HTML, CSS, and vanilla JavaScript.

## Live Preview

> Open `index.html` in your browser — no build step, no dependencies.

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Homepage with hero section, destination cards, and photo strip |
| `destinations.html` | Full guides for 10 destinations — best time, attractions, food |
| `gallery.html` | Photo grid of 20 landscapes across India |
| `travel-tips.html` | Practical tips — packing, budget, safety, food, trains |
| `contact.html` | Contact form with social links |

## Destinations Covered

- Jaipur — The Pink City
- Goa — The Golden Coast
- Varanasi — The Eternal City
- Kerala — God's Own Country
- Leh-Ladakh — The Last Frontier
- Rishikesh — Yoga Capital of the World
- Hampi — Ruins of a Lost Empire
- Rann of Kutch — The Great White Desert
- Andaman Islands — Emerald Seas at World's Edge
- Meghalaya — The Abode of Clouds

## Tech Stack

- **HTML5** — semantic markup throughout
- **CSS3** — custom properties, grid, flexbox, animations
- **JavaScript** — vanilla JS for navbar, smooth scroll, form validation
- **No frameworks, no dependencies**
